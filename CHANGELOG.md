# Changelog

## [0.1.2] – Stable – Grand Staff Enhanced Edition
### Added
- Improved Chord Detection engine with Jazz harmony support
- Enhanced Sound Engine with smoother playback
- Full Grand Staff Previewer with improved engraving
- Integrated Chord Previewer linked to the 88-key layout
- Full 88-Key Piano layout with synchronized staff feedback
- MIDI Loader module added for future import support
- Fullscreen Mode for distraction-free performance
- Updated Bravura notation fonts and FONTLOG
- Expanded MuseScore-friendly MusicXML structures

### Changed
- Better alignment of notation rendering
- Refined playback timing between staff, keyboard, and sound engine
- Stronger MIDI In/Out reliability and filtering
- Smoother UI flow across staff and playback modules

### Fixed
- Playback desync issues
- Incorrect chord naming on quick input
- Stuck notes from rapid MIDI device events
- Rendering bugs on narrow screens and mobile
- Cleaner layout scaling at different zoom levels

### Known Issues
- MusicXML import is still missing
- MIDI export remains experimental
- Staff beams occasionally misalign

### Roadmap
- MIDI loading into staffs
- MIDI and MusicXML export improvements
- Multi-track staff rendering
- Performance quantization and left-hand rhythm detection
